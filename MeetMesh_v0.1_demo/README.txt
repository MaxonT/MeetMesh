# MeetMesh v0.1 (static demo)

A lightweight, offline When2Meet-style availability planner. No backend. Double-click `index.html` to open.

## Features
- Create poll: title, date range, daily hours, slot length (15/30/60 min), timezone label.
- Paint availability with click-drag; touch supported.
- Local participants via localStorage (keyed by your name).
- Heatmap overlap and Top Suggestions.
- Share/import with a one-line base64 code (no servers).
- Add sample participants for quick demo.

## Notes
- This is a **static demo** intended for quick testing.
- Data stays in your browser (localStorage).
